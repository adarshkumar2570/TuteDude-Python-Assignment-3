# Python Calculator Tasks

This repository contains two basic Python programs demonstrating fundamental concepts like loops, functions, and usage of built-in modules.

---

## Task 1: Factorial Calculator

This program calculates the factorial of a number entered by the user.

### Features
- Uses recursion to compute factorial
- Returns and prints the result

### File
factorial_calculator.py

### Example
Enter a number: 5  
The factorial is: 120

---

## Task 2: Math Module Calculations

This program uses Python's built-in `math` module to perform different mathematical operations.

### Features
- Square root calculation
- Logarithmic value
- Sine value

### File
math-module-task/math_module_for_calculations.py

### Example
Enter a number: 25  
Square root: 5.0  
Logarithm: 3.2188758248682006  
Sine: -0.13235175009777303
